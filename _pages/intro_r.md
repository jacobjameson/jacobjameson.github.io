# About This Course

This page contains the content for the "Introduction to Data Analysis in R" mini-course. The goal is to introduce R programming concepts with a focus on preparing and analyzing data, and conducting statistical simulations. We will cover how to read data into R, manipulate data with a suite of tools from the tidyverse package dplyr. We will also discuss some basic programming concepts including data types and operators.

This mini-course is aimed at people with no programming background that are looking for a gentle and practical introduction to data analysis in R. 

Each Module involves:

- a pre-recorded lecture with guided practice exercises</li>
- a corresponding lab to practice the skills taught in the video and stretch your thinking</li>

The labs are an essential part of the learning process as they walkthrough the use of many key functions and topics that are not explicitly covered in the videos.

## Module 0

**Installation Guide**


<p style="color:blue;">Mac Users</p>
To Install R:
  
<ol>
<li>Open an internet browser and go to <a href="https://www.r-project.org">www.r-project.org</a>.</li>
<li>Click the "download R" link in the middle of the page under "Getting Started."</li>
<li>Select a CRAN location (a mirror site) and click the corresponding link.</li>
<li>Click on the "Download R for (Mac) OS X" link at the top of the page.</li>
<li>Click on the file containing the latest version of R under "Files."</li>
<li>Save the .pkg file, double-click it to open, and follow the installation instructions.</li>
<li>Now that R is installed, you need to download and install RStudio.</li>
</ol>

To Install RStudio:
  
<ol>
<li>Go to <a href="https://www.r-project.org">https://www.rstudio.com</a> and click on the "Download" button on the top right ( right above "Pricing").</li>
<li>Click on "Download RStudio Desktop."</li>
<li>Click on the version recommended for your system, or the latest Mac version, save the .dmg file on your computer, double-click it to open, and then drag and drop it to your applications folder.</li>
</ol>

<p style="color:blue;">Windows Users</p>
To Install R:
  
<ol>
