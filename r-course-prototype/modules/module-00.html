Local-first setup path

Setup without mystery

Before learning R, learners need one clean local workflow.

This module helps beginners install R and RStudio, understand what a project folder is, and run one script locally. The browser cell gives an immediate win while local setup is still in progress.

  • Install R from the R Project and RStudio Desktop from Posit.
  • Identify the Console, Source, Environment, Files, Plots, and Help panes.
  • Create a folder for course work and save a first `.R` script.

Try in Browser

Run one line before installing anything.

The browser sandbox is here to lower the first barrier. It is not a replacement for RStudio; it is a safe place to see what code and output feel like.

First output

Print the exact phrase `Hello from R`.

Use quotation marks inside `print()`: `print("Hello from R")`.

Run or check your code.
Do in RStudio

Repeat the win locally.

  1. Download R from r-project.org.
  2. Download RStudio Desktop from posit.co/downloads.
  3. Create a folder named `learning-r` somewhere you can find again.
  4. Open RStudio, create a new script, type `print("Hello from RStudio")`, and run it.
  5. Save the script as `module-00-first-script.R` inside your course folder.
