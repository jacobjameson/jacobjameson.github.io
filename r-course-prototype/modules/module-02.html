Concepts

The file-to-question workflow has four moves.

Load tools

`library(pkg)` attaches package functions for the current session.

Read data

`read.csv()` turns a CSV file into a data frame object.

Inspect shape

`dim()`, `names()`, `head()`, and `summary()` reveal what you have.

Ask a question

Indexing and filters pull out the rows and columns that matter.

Mini-Project

College access scan

Scenario: A policy analyst wants a quick scan of college characteristics before deciding what questions to ask next.

Dataset: `College.csv` from the Module 2 materials.

Task: Read the data, clean the college-name column, summarize `Outstate`, and count how many schools have `Top10perc > 50`.

Interpretation: Write two sentences: one describing the tuition distribution and one describing the threshold count.

Future Rmd submission: An Rmd with setup code, import code, summary output, and the two-sentence interpretation.
