Learning model

The course is a loop, not a lecture archive.

Watch

Short segment placeholders define exactly what each future video should teach.

Try

Guided browser cells let learners run code, get hints, reset, and check work.

Transfer

Every module asks learners to repeat the skill in local RStudio with files and scripts.

Build

Mini-project labs use public policy and education data to create interpretable outputs.
